Tomasz Gil website
==================

Project of my personal website.
Available soon on [tomaszgil.pl](http://tomaszgil.pl/)

# Instalation

1. Install jekyll
```
gem install jekyll
```
2. Install missing dependencies
```
npm install
```
3. Run gulp
```
gulp
```
Server will run at localhost:3000.
